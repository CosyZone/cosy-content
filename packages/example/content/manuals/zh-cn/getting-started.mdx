# {frontmatter.title}

本指南将帮助您在 5 分钟内快速上手我们的产品。

## 前置要求

在开始之前，请确保您的系统满足以下要求：

- Node.js 18.0 或更高版本
- npm 或 yarn 包管理器
- 现代浏览器（Chrome、Firefox、Safari、Edge）

## 安装

### 使用 npm

```bash
npm install our-product
```

### 使用 yarn

```bash
yarn add our-product
```

## 基本使用

### 1. 导入模块

```javascript
import { Product } from 'our-product';
```

### 2. 创建实例

```javascript
const product = new Product({
  apiKey: 'your-api-key',
  environment: 'development',
});
```

### 3. 开始使用

```javascript
// 基本功能示例
const result = await product.initialize();
console.log('产品初始化成功！', result);
```
