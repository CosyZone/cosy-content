# {frontmatter.title}

本指南将详细介绍如何安装和配置我们的产品。

## 系统要求

### 最低要求
- **操作系统**: Windows 10, macOS 10.15+, Ubuntu 18.04+
- **内存**: 4GB RAM
- **存储**: 2GB 可用空间
- **网络**: 稳定的互联网连接

### 推荐配置
- **操作系统**: Windows 11, macOS 12+, Ubuntu 20.04+
- **内存**: 8GB RAM 或更多
- **存储**: 5GB 可用空间
- **网络**: 高速互联网连接

## 安装方法

### 方法一：使用包管理器

#### Windows (Chocolatey)
```powershell
choco install our-product
```

#### macOS (Homebrew)
```bash
brew install our-product
```

#### Linux (Snap)
```bash
sudo snap install our-product
```

### 方法二：手动安装

1. **下载安装包**
   ```bash
   wget https://github.com/example/our-product/releases/latest/download/our-product-latest.tar.gz
   ```

2. **解压文件**
   ```bash
   tar -xzf our-product-latest.tar.gz
   cd our-product
   ```

3. **运行安装脚本**
   ```bash
   ./install.sh
   ```

### 方法三：Docker 安装

```bash
docker pull our-product/our-product:latest
docker run -d --name our-product our-product/our-product:latest
```

## 配置步骤

### 1. 初始化配置

```bash
our-product init
```

### 2. 设置 API 密钥

```bash
our-product config set api-key YOUR_API_KEY
```

### 3. 配置环境

```bash
our-product config set environment production
```

## 验证安装

运行以下命令验证安装是否成功：

```bash
our-product --version
our-product status
```

## 故障排除

### 常见问题

#### 问题 1：权限错误
**错误信息**: `Permission denied`
**解决方案**: 
```bash
sudo chmod +x /usr/local/bin/our-product
```

#### 问题 2：依赖缺失
**错误信息**: `Library not found`
**解决方案**:
```bash
# Ubuntu/Debian
sudo apt-get install libssl-dev

# macOS
brew install openssl
```

#### 问题 3：网络连接问题
**错误信息**: `Connection timeout`
**解决方案**:
1. 检查网络连接
2. 配置代理设置
3. 尝试使用镜像源
